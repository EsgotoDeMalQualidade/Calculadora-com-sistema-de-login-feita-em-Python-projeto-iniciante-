# Calculadora com Login

Projeto iniciante em Python: um sistema de login simples que dá acesso a uma calculadora com operações básicas.

## Funcionalidades

- Tela de login (usuário e senha)
- Menu com 4 operações matemáticas: adição, subtração, multiplicação e divisão
- Tratamento de erro para divisão por zero
- Opção de sair da calculadora a qualquer momento

## Como rodar

1. Tenha o Python 3 instalado na sua máquina
2. Baixe o arquivo `calculadora.py`
3. No terminal, rode:

```
python calculadora.py
```
